# 管理员后台指南

管理员通过浏览器访问服务端域名进入 Web 后台。所有管理接口需 admin 角色。

## 首次配置

首次访问时若系统尚无管理员，会进入**首次配置向导**：填写管理员用户名、邮箱、密码，提交后自动登录。此后该入口关闭，访问根路径将进入登录页。

## 登录与账号安全

- 账号密码登录；连续失败会触发指数退避锁定。
- 首次由管理员创建/重置的账号会被标记为「需修改密码」，用户首次登录后必须改密。
- 在「账号设置」可修改自己的密码（修改后所有会话失效，需重新登录）。

## 用户管理（/users）

- **创建用户**：填用户名 + 邮箱，可自动生成 12 位强密码。创建后弹出接入链接 + 初始密码，一键复制发给员工。
- **列表**：分页、按用户名/邮箱模糊搜索、按状态筛选。
- **重置密码**：生成新强密码并撤销该用户全部会话，新密码一次性显示。
- **禁用 / 启用**：禁用后该用户无法登录且现有会话被撤销。
- **删除**：在事务内删除用户及其会话（节点级联清理待真实数据面接入后完善）。
- 管理员不能删除或重置自己的账号（请用「账号设置」改密）。

## 节点管理（/peers）

- 列表展示所有节点：状态色点（在线/离线/已下线）、设备名、所属用户、虚拟 IP、最后心跳（相对时间）、公网 endpoint。每 10 秒自动刷新。
- 支持按用户名/设备名搜索、按状态筛选。
- **强制下线**：移除服务端 WireGuard 配置并将节点标记为 `force_removed`，该节点下次心跳会被拒绝（提示重新登录）。

## 审计日志（/audit-logs）

- 记录所有写操作（POST/PATCH/DELETE）与登录成功/失败事件，含用户、动作类型、资源、IP、UA、状态码、时间。
- 支持时间范围、动作类型、用户名筛选；筛选条件同步到 URL，便于分享。
- 展开某行可查看完整 `metadata`。
- 日志按 `VPN_AUDIT_RETENTION_DAYS`（默认 180 天）自动清理。

## 仪表盘（/dashboard）

- 4 个 KPI：用户总数、在线节点、离线节点、今日新增节点。
- 离线超过 10 分钟的节点会显示告警，可一键跳转处理。
- 展示最近活动的节点列表与服务端系统信息（版本、子网、端口、服务端公钥、启动时间）。
